# 시장 배경

- 국내 데이터 운영 인건비는 연 12% 속도로 증가하고 있습니다
- 진단 리포트 1건 작성에 평균 3.6시간이 소요됩니다

# Data Clinic 개요
## 한 번의 업로드로 Diagnosis Report 가 자동 생성됩니다
- 진입은 파일 업로드 한 번으로 끝납니다
- 스키마 추정과 결측 패턴 분석은 모델이 자동 수행합니다
- 산출물은 PDF Diagnosis Report 와 인터랙티브 대시보드 두 형태로 제공됩니다
