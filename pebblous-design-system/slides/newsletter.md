## 부록 A. 작성 체크리스트

뉴스레터를 발송하기 전에 한 번 더 확인합니다.

### 카피 / 톤
- [ ] 한국어 본문이 `-ㅂ니다 / -습니다` 격식체인가?
- [ ] 자기 지칭(`저희`)을 남발하지 않았는가?
- [ ] 제품명을 `Data Clinic` (두 단어)로 표기했는가?
- [ ] 진단 결과물을 `Diagnosis Report`로 부르고 있는가?
- [ ] 핵심 수치(1시간 / 5% / 80% / x5 등)를 임의 변형 없이 사용했는가?
- [ ] 큰 숫자가 약식 단위(`26만 달러`, `3억 6천만 원`)로 표기되어 있는가?

### 시각 / 디자인
- [ ] 메인 컬러가 Pebblous Orange `#F86825` 인가? (임의 액센트 컬러 도입 X)
- [ ] 그라데이션 벽 / AI 글로시 3D / 스톡 포토가 들어가지 않았는가?
- [ ] CTA 버튼이 시각적으로 명확한가? (Orange 솔리드 + 흰색 텍스트)
- [ ] 이모지가 무분별하게 사용되지 않았는가? (`→`, `✓` 같은 유니코드 픽토그램은 절제해서 허용)

### 구조
- [ ] 메일 제목과 프리헤더가 후킹력 있는가?
- [ ] 문제 → 해결책 → 가이드 → CTA 흐름이 끊기지 않는가?
- [ ] CTA가 명확하고, 한 뉴스레터 내 핵심 CTA는 1~2개로 절제되어 있는가?
- [ ] 모든 외부 링크에 UTM 파라미터가 붙어 있는가?
- [ ] 모바일에서도 읽기 좋은 길이인가? (총 800~1,200자 권장, 5~7분 분량)

### 발송 / 운영
- [ ] 수신거부 링크가 동작하는가?
- [ ] 모든 외부 링크가 클릭 가능하고 의도한 URL로 연결되는가?
- [ ] 이미지에 alt 텍스트가 들어 있는가?
- [ ] 미리보기를 데스크톱 / 모바일 / 다크 모드에서 모두 확인했는가?
- [ ] Stibee 테스트 발송으로 본인/검수자가 직접 받아 봤는가?

  ↓ 섹션 2: 해결책 제시
  "가상 세계, 디지털 트윈 속에서 먼저 시행착오를 거친 후 현실에 적용"

    2단계 데이터 수집 및 의미 연결 — 센서 / 합성데이터

  ↓ CTA 1
  "→ 디지털 트윈 구축 7단계, 블로그에서 이어서 확인하세요"

  ↓ 섹션 4: 웨비나
  "단 20석 남은 웨비나, 지금 신청하기"
  마감일 4월 29일 강조
